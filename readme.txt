== Description ==

APM Child is a child plugin that enables APM members to link their
 APM Child Wordpress website with the APM Master plugin database.

APM Child enables APM members to link their APM Child Wordpress
 website with their ClickMagick.com account. Only statistical data and
 no personal information from website visitors is forwarded to ClickMagick.com

APM Child plugin provides a method to create and store Aweber.com API Key.


APM Child plugin provides a method to create and store JVZoo.com Affiliate 
Account details to create site-wide customised affiliate links.

APM Child creates post snippets using tags to assist in the personalization 
of the APM Child Wordpress website.

APM Child saves ALL the webmaster data entered directly into
 the APM Child plugin settings into the APM Master database. 

No APM Child Wordpress website visitor data is stored, forwarded or 
otherwise transmitted to the APM Master database.

The Master APM database is located on the following domain:
http://affiliatepromembership.com


== Installation == 
 
1. Install the plugin either by uploading it via FTP or by using the "Add Plugin" function of WordPress. 
2. Activate the plugin at the plugin administration page.
3. After Activation, APM Child will appear into left Menu section.
4. Just go to APM Child and click on Login button, it will automatically log you in and then you will get access to the plugin settings.
